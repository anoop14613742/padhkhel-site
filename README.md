★ SCHOOL LEARNING (Play Group to Class 12)
Browse the way that suits you — by age, by class, or by subject. Practise English, Maths, EVS, Science, Hindi, Arts, Sports & Yoga and more with fun activity types:
• Picture matching — great for early learners
• Multiple-choice questions with instant feedback and explanations
• Fill-in-the-blanks to build recall
